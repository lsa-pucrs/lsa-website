Sobre o LSA

Integrando disciplinas para robótica

O Laboratório de sistemas autônomos reúne pesquisadores de diversas áreas de pesquisa, desde o desenvolvimento do hardware robótico até o controle de comportamento de alto nível. O laboratório conta com uma infra-estrutura completa para a implementação de robôs móveis de pequeno porte, incluindo diversas plataformas robóticas padrão na pesquisa atual, em particular contamos com diversas plataformas programáveis utilizando o Robot Operating System.
